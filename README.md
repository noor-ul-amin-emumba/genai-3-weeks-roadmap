# genai-3-weeks-roadmap

## Overview

This repository tracks my progress through a 3-week GenAI learning roadmap. Work is organized by week and day, with task files and a short report per day.

## Structure

- `week-1/day-1/`
- `week-1/day-2/`
- `week-2/day-1/`
- `week-3/day-1/`

## Notes

- Each day includes scripts, outputs, and a report file.
- LLM usage is implemented via the Groq API.
